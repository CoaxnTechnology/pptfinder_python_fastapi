Internship Project

Developed By: Sahad Shaikh
Project Type: Full Stack Web Application (Backend)
Technology: FastAPI (Python)
Database: PostgreSQL

Project Description

This is the backend part of the PPT Finder internship project.
The backend is built using FastAPI and provides REST APIs to handle PPT search requests, API integration, and database operations.

Search results are stored and managed using a PostgreSQL database.

Features

FastAPI REST API

Keyword-based PPT search endpoint

API integration for fetching PPT data

PostgreSQL database storage

SQLAlchemy ORM

Clean and scalable backend structure

Technologies Used

FastAPI

Python

PostgreSQL

SQLAlchemy

Project Purpose

This backend was created to:

Learn FastAPI backend development

Understand REST API architecture

Work with PostgreSQL databases

Gain real-world internship experience
